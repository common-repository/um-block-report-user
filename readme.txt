Block any user from viewing your profile page or report any user to admin. Toxic user account will be automatically deactivated after getting blocked or reported frequently. 

== Description ==

This Ultimate Member extension will allow you to block any toxic user. Blocked users can't view your public profile or you in member directory. Users will see
Block & Report button in other user's profile and take action if necessary.

Official Developer: [Mansur Ahamed](https://www.upwork.com/freelancers/~013259d08861bd5bd8)

= Key Features: =

* Report any user to admin
* Getting 10 block or 20 report will result toxic user account deactivation automatically
* Add profile tabs to view your blocked and reported list
* Admin can see in Users menu who is getting blocked and reported
* Allows to unblock user
* Admins can't be blocked
* Remove user from member directory listing for blocked user

== Installation ==

All you'll need to do to get this setup is:

1. Install and Activate the plugin
2. Admin can go to Users menu inside dashboard to check the Block and Report Column where number of block and report listed.

That's it!
